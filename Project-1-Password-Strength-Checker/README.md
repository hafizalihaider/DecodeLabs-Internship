# Password Strength Checker

## Description
A Python-based password strength checker developed during the DecodeLabs Cyber Security Internship.

## Features
- Password length validation
- Uppercase detection
- Lowercase detection
- Digit detection
- Special character detection
- Space detection
- Sequential pattern detection
- Password strength classification (Weak, Medium, Strong)
- Detailed remarks

## Technologies
- Python 3.14
